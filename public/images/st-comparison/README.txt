Image naming for the "digitalizar operador turístico" comparison page.

Expected filenames (N = row number 1-5):
  N-wo-st.jpg   = "Without" Simple Travel (Sin ST) for row N
  N-w-st.jpg    = "With" Simple Travel (Con ST) for row N

Examples:
  1-wo-st.jpg, 1-w-st.jpg   (row 1: Cotizaciones)
  2-wo-st.jpg, 2-w-st.jpg   (row 2: Tarifas y prestadores)
  3-wo-st.jpg, 3-w-st.jpg   (row 3: Control de márgenes)
  4-wo-st.jpg, 4-w-st.jpg   (row 4: Liquidaciones y pagos)
  5-wo-st.jpg, 5-w-st.jpg   (row 5: Trabajo en equipo)

You may use other names in the middle (e.g. 1-cotizaciones-wo-st.jpg) only if
you rename the references in the Blade view to match, or use the short form
above. Images are generic and language-independent.
